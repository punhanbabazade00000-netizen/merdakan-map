      Punish MMC Giriş Paneli

      Admin və adi istifadəçilər üçün ayrı-ayrı giriş seçimləri. Sistemi idarə etmək və məlumatları görüntüləmək üçün müvafiq girişi seçin.

      Admin Giriş

      Sistem idarəçiliyi, məlumat əlavəsi/silənməsi, istifadəçi rollarının tənzimlənməsi və analiz üçün admin panelə giriş.

      Admin girişi üçün məlumatları sistem administratoru təqdim edir.

      İstifadəçi Giriş

      Punish MMC məlumatlarını görüntüləmək, axtarmaq və şəxsi hesabı idarə etmək üçün istifadəçi panelə giriş.

      Şifrəni unutmusunuz?

      İstifadəçi qeydiyyatı üçün əlavə "Qeydiyyat" seçimi mövcuddur.
